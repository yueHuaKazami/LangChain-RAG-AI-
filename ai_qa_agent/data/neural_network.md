# 神经网络基础

## 神经元模型

神经网络的基本单元是神经元，模仿生物神经元的工作方式。

- **输入**：x₁, x₂, ..., xₙ
- **权重**：w₁, w₂, ..., wₙ（每个输入对应一个权重）
- **偏置**：b
- **加权求和**：z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
- **激活函数**：a = f(z)，引入非线性

## 网络结构

### 前馈神经网络

信息从输入层单向传播到输出层，经过若干隐藏层。

- **输入层**：接收特征
- **隐藏层**：提取特征，进行非线性变换
- **输出层**：给出预测结果

### 深度神经网络

含多个隐藏层的网络。"深度"指层数多。深度网络能学习更复杂的层次化特征。

## 前向传播

数据从输入层逐层计算到输出层的过程：

1. 输入层接收数据
2. 每一层计算：z = Wx + b（线性变换）
3. 应用激活函数：a = f(z)（非线性变换）
4. 将结果传给下一层
5. 输出层给出预测值

## 反向传播

训练神经网络的核心算法，用于计算损失函数对各参数的梯度。

1. **前向传播**：计算预测值
2. **计算损失**：用损失函数衡量预测值与真实值的差距
3. **反向传播**：从输出层向输入层，用链式法则逐层计算梯度
4. **参数更新**：用梯度下降更新权重和偏置

## 激活函数

引入非线性，使网络能学习复杂的非线性关系。

### Sigmoid

f(x) = 1/(1+e⁻ˣ)，输出范围 (0,1)。缺点：梯度消失、输出非零中心。

### Tanh

f(x) = (eˣ-e⁻ˣ)/(eˣ+e⁻ˣ)，输出范围 (-1,1)。比 Sigmoid 好，但仍可能梯度消失。

### ReLU

f(x) = max(0, x)，计算简单、收敛快。是最常用的激活函数。缺点：负输入梯度为 0（神经元死亡）。

### Leaky ReLU

f(x) = max(αx, x)，解决神经元死亡问题，α 通常取 0.01。

### GELU

高斯误差线性单元，Transformer 中常用。f(x) = x·Φ(x)，Φ 为标准正态分布的累积分布函数。

## 损失函数

衡量预测值与真实值的差距，训练的目标是最小化损失。

### 均方误差（MSE）

L = (1/n)Σ(ŷᵢ - yᵢ)²，用于回归任务。

### 交叉熵损失

用于分类任务。二分类：L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]。多分类：L = -Σyᵢ·log(ŷᵢ)。

## 优化算法

### 梯度下降

沿损失函数的负梯度方向更新参数：θ = θ - η·∇L。η 为学习率。

### 随机梯度下降（SGD）

每次用一个或少量样本更新参数，速度快但有噪声。

### Adam

自适应学习率优化器，结合动量和自适应学习率，是深度学习中最常用的优化器。

## 卷积神经网络（CNN）

擅长处理图像等网格结构数据。

- **卷积层**：用卷积核提取局部特征，参数共享
- **池化层**：降采样，减少参数和计算量
- **全连接层**：整合特征做分类

经典网络：LeNet、AlexNet、VGG、ResNet、Inception

## 循环神经网络（RNN）

擅长处理序列数据（文本、时间序列）。

- 隐藏状态在时间步之间传递，具有记忆能力
- 缺点：长序列梯度消失/爆炸

### LSTM（长短期记忆）

通过门控机制解决梯度消失问题：
- **遗忘门**：决定丢弃哪些信息
- **输入门**：决定存入哪些新信息
- **输出门**：决定输出什么

### GRU

LSTM 的简化版，合并了部分门控，计算更高效。

## 注意力机制

允许模型在处理序列时动态关注不同位置，是 Transformer 的核心组件。

- 自注意力：序列内每个位置与所有位置计算关联
- 多头注意力：多组注意力并行计算，捕捉不同维度的信息
